# Collatz Halving Depth Analysis

This repository contains the Python code used in the paper:

"Structural Drivers of Growth in Collatz Trajectories: A Quantitative Analysis of Halving Depth and Volatility"

## Files

- parity_analysis.py  
  Generates Table 1 and Figures 1–2 (parity-based analysis)

- halving_depth_analysis.py  
  Generates Figures 3–4 and statistical analysis (halving depth analysis)

## Data

All data is generated programmatically for integers:
1 ≤ n ≤ 100,000

## Requirements

- Python 3
- numpy
- matplotlib

## Notes

Results match those reported in the paper, with minor rounding differences.

## Reproducibility

Running the scripts will regenerate all figures and results reported in the paper.
